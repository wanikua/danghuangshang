```
遵命！

【问题1：SQL注入】

❌ 原代码：
def login(username, password):
    sql = f"SELECT * FROM users WHERE 
           username='{username}' AND 
           password='{password}'"
    return db.execute(sql)

✅ 修复后：
def login(username, password):
    sql = "SELECT * FROM users WHERE 
           username=? AND password=?"
    return db.execute(sql, (username, 
           hash_password(password)))

【问题2：内存泄漏】

❌ 原代码：
def process_file(filepath):
    data = open(filepath).read()  # 全读
    return parse(data)

✅ 修复后：
def process_file(filepath):
    with open(filepath) as f:
        for line in f:  # 流式读取
            yield parse(line)

【问题3：并发Bug】

❌ 原代码：
class Cache:
    def get(self, key):
        return self.data[key]
    def set(self, key, val):
        self.data[key] = val

✅ 修复后：
import threading

class Cache:
    def __init__(self):
        self.data = {}
        self.lock = threading.Lock()
    
    def get(self, key):
        with self.lock:
            return self.data.get(key)
    
    def set(self, key, val):
        with self.lock:
            self.data[key] = val

【优化方案：N+1查询】

❌ 原代码：
users = User.all()
for user in users:
    user.posts = Post.find(user.id)  # N次查询

✅ 修复后：
users = User.all()
posts = Post.where(user_id__in=[u.id for u in users])
# 只查1次，用字典映射
```
