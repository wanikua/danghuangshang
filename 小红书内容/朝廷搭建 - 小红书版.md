## 第一步：安装工具⚙️（1分钟）

打开电脑终端（Mac）或命令提示符（Windows）

**复制粘贴这行代码：**
```bash
npm install -g clawdbot
```

**看到安装完成就OK了！** ✅

> 💡 不会安装？
> 1. 先去 https://nodejs.org 下载Node.js
> 2. 一路"下一步"安装
> 3. 再执行上面命令

---

## 第二步：创建朝廷🏛️（2分钟）

### 1. 创建目录
```bash
mkdir ~/my-dynasty
cd ~/my-dynasty
clawdbot init
```

### 2. 配置API密钥
```bash
clawdbot config set providers.anthropic.apiKey "sk-ant-你的密钥"
```

> 🔑 密钥哪里来？
> 1. 访问 https://console.anthropic.com
> 2. 注册账号
> 3. 创建API Key
> 4. 复制粘贴到上面命令

### 3. 创建5个核心文件

直接复制下面内容，创建对应文件：

**SOUL.md**
```markdown
# 性格设定
- 中文为主
- 直接说结论
- 废话少说
```

**IDENTITY.md**
```markdown
# 朝廷架构
👑 你 — 天子
⚡ 大内总管 — 日常对话
├─ ⚔️ 兵部 — 代码专家
├─ 💰 户部 — 财务专家
└─ 🎭 礼部 — 营销专家
```

**USER.md**
```markdown
# 关于你
- 名字：你的名字
- 语言：中文
```

**AGENTS.md**
```markdown
# 工作手册
1. 读SOUL.md（性格）
2. 读USER.md（主人）
3. 读memory（最近发生的事）
```

**TOOLS.md**
```markdown
# 可用工具
- 文件操作
- 网页搜索
- 命令执行
```

## 第三步：启动朝廷🚀（2分钟）

### 1. 编辑配置
```bash
clawdbot config edit
```

### 2. 粘贴配置
```json
{
  "agents": {
    "defaults": {
      "workspace": "~/my-dynasty",
      "model": "anthropic/claude-sonnet-4-5"
    },
    "list": [
      {
        "id": "main",
        "name": "大内总管",
        "default": true,
        "model": "anthropic/claude-sonnet-4-5"
      },
      {
        "id": "bingbu",
        "name": "兵部尚书",
        "model": "anthropic/claude-opus-4-5",
        "systemInject": "你是兵部尚书，专精代码审查、系统架构。"
      },
      {
        "id": "hubu",
        "name": "户部尚书",
        "model": "anthropic/claude-opus-4-5",
        "systemInject": "你是户部尚书，专精财务分析、成本管控。"
      },
      {
        "id": "libu",
        "name": "礼部尚书",
        "model": "anthropic/claude-sonnet-4-5",
        "systemInject": "你是礼部尚书，专精品牌营销、文案创作。"
      }
    ]
  },
  "providers": {
    "anthropic": {
      "apiKey": "sk-ant-你的密钥"
    }
  }
}
```

### 3. 启动！
```bash
clawdbot gateway start
clawdbot gateway status
```

看到 **✓ Running** 就成功了！🎉

## 开始使用🎯

### 日常对话
```bash
clawdbot chat "今天天气怎么样？"
# 大内总管回复（便宜💰）
```

### 召唤部门
```bash
# 代码审查 → 兵部
clawdbot chat --agent bingbu "审查这段代码"

# 财务分析 → 户部
clawdbot chat --agent hubu "分析这个月支出"

# 写文案 → 礼部
clawdbot chat --agent libu "写条小红书"
```
