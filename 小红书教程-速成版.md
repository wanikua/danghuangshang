## 🚀 3 步搭建（30 分钟搞定）

### 第 1 步：领免费服务器（10 分钟）

去 **Oracle Cloud** 注册账号，领一台**永久免费**的云服务器：

```
配置：4 核 24GB 内存 + 200GB 硬盘 + 10TB 流量
价值：≈$100/月，但 Oracle 免费送！
```

**关键点：**
- 选日本东京区域（容量足）
- 一定要选 ARM 架构（Ampere）
- 下载好 SSH 密钥（只此一次！）

> ❗ **易错点**：信用卡验证用 Visa/MasterCard，银联不行！

### ⚠️ 中国大陆用户特别说明（重要！）

**可能遇到的问题：**
- 🚫 中国大陆 IP 可能被限制注册（部分地区）
- 🚫 免费额度抢手，可能显示"Out of capacity"
- 🚫 需要国际信用卡（Visa/MasterCard）

**别担心！有 3 个解决方案：**

#### 🟢 方案 1：挂梯子注册（推荐⭐）
- 用香港/日本/新加坡节点
- 邮箱用 Gmail/Outlook
- **成功率 90%+**

#### 🟡 方案 2：用其他云服务商免费额度
- AWS 12 个月免费（需信用卡）
- Google Cloud $300 额度（需信用卡）
- 腾讯云/阿里云学生机（¥9.9/月）

#### 🟢 方案 3：本地运行（最简单⭐⭐⭐）
- Mac/Windows 直接装 Clawdbot
- 不用服务器，24 小时开机
- **成本：0 元（只付 API 费）**

> 💡 **建议**：方案 3 最适合新手！不用折腾服务器，5 分钟搞定！

### 第 3 步：配置朝廷架构（15 分钟）

#### 3.1 获取 API Key（3 分钟）

1. 访问 https://console.anthropic.com
2. 注册/登录
3. 点击"API Keys" → "Create Key"
4. **复制 Key**（格式：sk-ant-api03-xxxxx）

> ❗ **重要**：Key 只显示一次！立刻复制保存！

#### 3.2 配置环境变量（1 分钟）

```bash
echo 'export ANTHROPIC_API_KEY="sk-ant-api03-你的密钥"' >> ~/.bashrc
source ~/.bashrc
```

#### 3.3 创建朝廷配置文件（10 分钟）

在服务器创建 `~/clawd/IDENTITY.md`：

```markdown
# 我的 AI 朝廷

## 架构
👑 我（天子）
⚡ 大内总管 [Sonnet] — 日常对话 + 任务路由
├─ ⚔️ 兵部 [Opus] — 代码/技术
├─ 💰 户部 [Opus] — 财务/电商
└─ 🎭 礼部 [Qwen] — 营销/内容
```

创建 `~/.clawdbot/clawdbot.json`：

```json
{
  "agents": {

      "model": "anthropic/claude-sonnet-4-5"

    "list": [
      {
        "id": "main",
        "name": "大内总管",

      {

        "name": "兵部尚书",

        "systemInject": "你是兵部尚书，专精软件工程和系统架构。"
      },
      {
        "id": "hubu",
        "name": "户部尚书",
        "model": "anthropic/claude-opus-4-5",
        "systemInject": "你是户部尚书，专精财务分析和电商运营。"

      {
        "id": "libu",
```

#### 3.4 启动朝廷（1 分钟）

```bash
clawdbot gateway start
clawdbot gateway status
```

## ❓ 常见问题 Q&A

**Q: Oracle 注册一直失败怎么办？**

A: 
- 检查是否用了国内 IP（换梯子节点）
- 检查信用卡类型（必须 Visa/MasterCard）
- 换个邮箱试试（Gmail 最稳）
- 实在不行用方案 2/3

**Q: 显示"Out of capacity"怎么办？**

A:
- 换其他区域（ap-tokyo-1 / ap-seoul-1）
- 换时间段（凌晨美国时间）
- 等几天再试
- 用其他云服务商
