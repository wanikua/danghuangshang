## 2. Description of the Project

The Project is an open-source, multi-agent AI collaboration system built on the [OpenClaw](https://github.com/openclaw/openclaw) framework. It provides configuration templates, installation scripts, a GUI dashboard, and Docker images for deploying AI agents organized in a traditional Chinese government structure ("三省六部制").
