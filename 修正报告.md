## 🔧 修正内容详情

### 1. 删除不存在的命令

#### ❌ 错误示例
```bash
clawdbot chat "你好，自我介绍一下"
clawdbot chat --agent bingbu "用 Python 写个 Hello World"
clawdbot config edit
clawdbot config set xxx "value"
clawdbot gateway logs
```

#### ✅ 正确替换
```bash
# Clawdbot 通过 Discord/Signal/Telegram 交互，不是命令行工具
在 Discord 里 @司礼监 说"你好，自我介绍一下"
在 Discord 里 @兵部尚书 说"用 Python 写个 Hello World"
nano ~/.clawdbot/clawdbot.json
直接编辑 ~/.clawdbot/clawdbot.json 文件
cat /tmp/clawdbot/clawdbot-$(date +%Y-%m-%d).log
```

### 2. 修正 agent 配置格式

#### ❌ 错误格式
```json
{
  "id": "bingbu",
  "name": "兵部尚书",
  "model": "anthropic/claude-opus-4-5",
  "systemInject": "你是兵部尚书...",
  "default": true
}
```

#### ✅ 正确格式
```json
{
  "id": "bingbu",
  "model": {
    "primary": "anthropic/claude-opus-4-6"
  },
  "identity": {
    "name": "兵部尚书",
    "theme": "你是兵部尚书...",
    "emoji": "⚔️"
  }
}
```

### 3. 修正模型名称

- ❌ `claude-opus-4-5`  
- ✅ `claude-opus-4-6`

### 4. 修正 Notion 配置方式

#### ❌ 错误方式
```bash
echo 'export NOTION_TOKEN="secret_xxx"' >> ~/.bashrc
```

#### ✅ 正确方式
```bash
mkdir -p ~/.config/notion
echo "secret_xxx" > ~/.config/notion/api_key
```

### 5. 修正 GitHub Token 权限

- ❌ `repo, workflow, admin:org`  
- ✅ `repo, workflow`

### 6. 删除假的一键安装脚本

- ❌ `https://raw.githubusercontent.com/wanikua/clawd/main/install-clawdbot.sh`  
- ✅ 分步手动安装说明

### 7. 修正安装命令

#### ❌ 错误
```bash
sudo npm install -g clawdbot
```

#### ✅ 正确
```bash
# 先安装 Node.js
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
source ~/.bashrc
nvm install 22
npm install -g clawdbot
```

### 8. 修正 Bot 提及

- ❌ `@Clawdbot`  
- ✅ `@司礼监` 或 `@对应部门名`

### 9. Shell 脚本中的 clawdbot chat 调用

将所有在 `.sh` 脚本中的 `clawdbot chat` 调用改为注释说明：
```bash

# 使用 Clawdbot cron 功能或在 Discord 里触发生成日报
```
